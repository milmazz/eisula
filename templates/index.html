Perfil del ingeniero

GNU Octave

Estamos desarrollando ejemplos de cálculo y simulación en GNU Octave mostrando gráficos, simulaciones de ecuaciones diferenciales y ejemplos de control de sistemas no lineales. Se pueden hacer modificaciones de los parámetros y, en el caso de ejemplos de sistemas de control, escoger las estrategias de control a utilizar. Los resultados se presentan en figuras del tipo PNG. Se presentan, por ejemplo, diagramas de Bode y de Nyquist para un sistema de segundo orden. El primer ejemplo de sistemas de control es el de un artefacto espacial controlado mediante linealización aproximada. Se presenta un control por realimentación del vector de estados, incluyendo un observador de orden completo que puede incorporarse o no en la simulación.
